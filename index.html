Summary: Latitude vs X

The purpose of the project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 citites.

After assembling the dataset, we used Matplotlob to plot various aspects of the weather vs latitude. We looked at the follwing factors: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
